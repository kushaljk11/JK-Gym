# JK-Gym

A simple Java Swing application for managing gym members (Regular and Premium). This project provides a GUI to add, update and persist member data.

## Prerequisites
- Java Development Kit (JDK) 8 or newer installed and `javac`/`java` available on PATH.
- (Optional) Git installed if you want to commit and push changes.

## Run from Command Line (Windows PowerShell)
1. Open PowerShell.
2. Change directory to the project folder:

```powershell
cd "c:\Users\Nts\OneDrive - London Metropolitan University\1st year\java cw all\JK-Gym"
```

3. Compile the Java sources:

```powershell
javac *.java
```

4. Run the application (the main class is `gymGUI`):

```powershell
java gymGUI
```

Notes:
- If you see errors about missing images (e.g. `8326767-02.png`), ensure the image file is in the same folder or update the path in `gymGUI.java`.
- If compilation fails due to missing classes, confirm all `.java` files (e.g., `GymMember.java`, `PremiumMember.java`, `RegularMember.java`) are present in the same folder before running `javac`.

## Run from an IDE
- Import the folder as a Java project in IntelliJ IDEA, Eclipse or VS Code (with Java extensions).
- Locate the `gymGUI` class and run it as a Java application.
